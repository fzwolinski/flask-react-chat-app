# 🏫 Flask React Chat App

**Flask React Chat App** - real time chat web chat app.

#### 🎨 Tech Stack

- **Backend**
  - Server - Flask
  - Sockets - Flask-SocketIO
  - Database - sqlite3 (Flask-SQLAlchemy)
- **Frontend**
  - Library - React
  - Sockets - socket.io-client
  - Design - Material-UI
